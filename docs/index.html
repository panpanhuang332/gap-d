ABOUT ME

數位時代的
期望誠實管理

致力於修復被假評污染的數位市場與品質期望

我是黄國津，建國科技大學經營管理系（進修部二技）學生，研究領域聚焦於服務品質管理、消費者行為、電子口碑（eWOM）與數位假評（Fake Reviews）偵測。

我的研究關注一個被既有服務品質框架忽略的問題：在純數位商務情境中，網路評論已大幅取代傳統的實體接觸媒介，成為消費者形成期望的主要資訊基礎。當這些評論遭假評系統性污染，消費者的期望往往在真正接觸服務之前，就已被悄然扭曲——而 PZB 的 SERVQUAL 量表長期聚焦於「服務後」的期望與感知落差（Gap 5），始終未處理「期望形成階段」本身是否真實的問題。

環繞這個缺口，我發展了「Gap D 三部曲」研究：第一部提出並論證「數位資訊污染缺口（Gap D）」與「期望扭曲悖論（EDP）」的現象與機制；第二部從資訊問責性、信號可信度與市場失靈三個維度，論證 Gap D 有別於 PZB 原始 Gap 4 的理論獨立性；第三部建構「EAQUAL-15 期望真實性量表」原型與「MACS」平台端配套機制，將理論命題轉化為可操作的測量與治理工具。三篇論文均已獲接受並受邀進行口頭發表（其中台北商業大學研討會論文已於 5/15 完成口頭發表）。

我的長期目標，是將上述構念整合為以「期望誠實管理（EHM, Expectation Honesty Management）」為主軸的服務品質管理框架——從消費者端的期望校準，到平台端的市場失靈治理，提供一套系統性的研究取徑。

INTERACTIVE TOOL B

EAQUAL-15 期望真實性自我評估問卷

本問卷基於作者第三篇論文《EAQUAL-15：期望真實性量表初步原型建構》設計。您可以選擇一則您剛剛看過的網路評論，並針對該則評論的來源可信度 (SC)、內容真實性 (CA)、體驗一致性 (EC) 進行主觀打分。系統會自動彙整出雷達分析圖與期望安全級別。

第一階段：來源可信度 (Source Credibility, SC)

評估該評論發布帳號的歷史活動成本與真實身分可追溯度

SC1：評論者提供的個人或帳號資訊，足以讓我判斷其評論可信度。

SC2：具有長期活動紀錄的評論者，比新建立帳號更值得信任。

SC3：這則評論的來源讓我懷疑其可能為業者或第三方刻意撰寫。反向題

SC4：評論者過去的發文紀錄，有助於判斷其評論可信度。

第二階段：內容真實性 (Content Authenticity, CA)

評估該評論內容是否包含豐富的真實體驗細節（MOT 特徵）

CA1：這則評論提到了具體的時間、地點或人員互動。

CA2：這則評論描述了具體的感官或服務細節。

CA3：這則評論描述了實際發生的消費過程。

CA4：這則評論包含許多具體而非籠統的描述。

CA5：這則評論只有情緒表達，缺乏具體細節。反向題

CA6：評論中的情緒表達符合一般消費者的說話方式。

第三階段：體驗一致性 (Experience Consistency, EC)

購後驗證階段 —— 評估您實際消費體驗與先前該評論描述的一致性

EC1：評論提到的服務細節，在實際消費後可以被確認。

EC2：我實際體驗到的服務內容，與評論描述大致一致。

EC3：部分評論內容與我實際體驗有明顯差異。反向題

EC4：我認為這些評論能真實反映實際消費情況。

EC5：消費後，我認為評論對服務的描述大致可信。

Gap D 延伸應用與口碑治理願景

類 Truecaller 主動式守門員

Proactive Gatekeeper

將目前的「被動式複製貼上分析 (v0.1)」演進為「主動式守門員 (Proactive Gatekeeper)」。本研究之長期願景為開發類似 Truecaller 的即時辨識系統。當消費者在瀏覽 Google Maps 或評論平台時，以瀏覽器擴充功能（Extension）形式在後台自動掃描，並於期望形成前，即時在頁面中呈現評論真實性風險標籤與健康度儀表板。

定位：消費者端 —— 體制內自動防護

MARC 真實評論認證

MOT Authentic Review Certification

服務業者端主動防禦機制。誠實經營的商家可主動申請認證，授權系統存取其歷史評論語料。系統透過 MOT 演算法排除虛假評價干擾後，計算並頒發「Gap D 還原後的真實星等（E_authentic Star Rating）」認證標記，以此高信賴度的第三方信用信號（Signaling Mechanism）發揮市場「良幣驅逐劣幣」的淨化效果。

預計原始工具開發時程表

2026/11/30 前：完成商家端 API 數據對接與認證演算核心系統測試。

定位：服務業者端 —— 信用信號傳遞

MACS 主動口碑新生態

MOT Active Consumer Scoring

引導消費者成為數位時代的「米其林神秘客」。本評分系統捨棄傳統的主觀給星與自由評語，改以點選「隨機抽取的 MOT 服務接觸細節題組」。評分時系統從題庫中隨機生成題組以進行人機驗證（對抗 AI 批量洗分），建立一個與現有平台隔離、未被 Gap D 資訊污染的乾淨口碑生態。

預計原始工具開發時程表

2026/07/31 前：完成 3000 則以上語意庫自各 LLM 萃取，以及 App 開發測試。

定位：消費者社群 —— 平行生態系統

核心治理哲學：被動式濾水器 vs 主動式挖新井

這三個延伸應用工具體現了從「個體防禦」走向「生態治理」的思維轉變：
被動式濾水器（MOT Calibrator v0.1）：是在已被假評污染的河流中，由消費者端裝設濾水器過濾雜質，屬於體制內的防護，仰賴使用者有意識地複製貼上。
主動式挖新井（MARC 認證 & MACS 評分）：則是直接在體制外另外挖一口乾淨的新井，藉由隨機接觸題組與第三方星等還原，引導誠信商戶與消費者建立完全無污染的平行信號生態。
